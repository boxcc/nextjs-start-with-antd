# [Next.js](https://nextjs.org/) 的 [ant-design](https://github.com/ant-design/ant-design) 启动模板
## 主要依赖列表
- Next.js
- ant-design
- Typescript
- react-query
- graphql
- postcss
- tailwindcss

等等...

## 运行
```
pnpm install
pnpm dev
```

## 注意事项

本地需要安装 [gitflow-avh](https://github.com/petervanderdoes/gitflow-avh/wiki/Installation)才能使用命令
```
pnpm release
```
